# Aplikacija za pretragu letova

Aplikacija je izradjena tokom odrzavanja prakse studentima u Racunarskom centru Univerziteta Singidunum

## Postavka zadtaka

Napraviti aplikaciju koristeci Vue 3 framework u JavaScript jeziku. Aplikacija je potrebno da komunicira za vec postojecim backendom za pretragu letova. Aplikacija mora da sadrzi prikaz po stranicama i prikaz detalja leta. U prikazu detalja leta je potrebno prikaziti trenetnu vremensku prognozu na datoj lokaciji, pun naziv lokacije, sliku lokacije kao i prikaz na google mapi. Dodatno je potrebno implementirati pretragu letova po destinaciji uz pomoc autocomplete funkcije u polju za unos. Sve pozive ka eksternim podacima je potrebnom implemnetirati uz pomoc servisnih klasa. Potrebno je napraviti odvojene komponente za jedan red tabele prikaza jednog leta kao i komponentu za prikaz animacije ucitavanja podataka.

Takvu napravljenu aplikaciju je potrebno pustiti u produkciju uz pomoc GitHub Pages infrastukture u okviru GitHub platforme. Studenti moraju da naprave repozitorijum i da uz pomog alata GitHub Desktop sve izmeme u kudu postavljaju na GitHub u javno dostupni repozitorijum

## Zahtevi po stavkama

- Koriscenje Vue 3 + JavaScript okruzenja
- Upotreba Vue Routera
- Boostrap 5 ili drugi CSS framework
- Pozivanje eksternog API za podatke
- Upotreba Axios bibliteke
- Prikaz mape uz pomoc Google Maps Embeda
